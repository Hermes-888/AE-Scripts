Thank you for downloading the Fast Marker Creator script for Adobe After Effects. To install this script place it in your ScriptUI Panels folder for the After Effects version you are using. Once installed, you can run the script from either navigating to the Window menu and scrolling down to click on "FastMarkerCreator_v3", or by navigating to the File menu, then Scripts and clicking on "FastMarkerCreator_v3".

http://markertriggery.com/
https://www.youtube.com/watch?v=5HQSyhV7wFI


The script install default paths are...
     MAC:
	(dockable panel)
	Master HD/Applications/Adobe After Effects <version>/Scripts/ScriptUI Panels/

	(floating panel)
	Master HD/Applications/Adobe After Effects <version>/Scripts/

     PC:
	(dockable panel)
	C:\Program Files\Adobe\Adobe After Effects <version>\Support Files\Scripts\ScriptUI Panels\

	(floating panel)
	C:\Program Files\Adobe\Adobe After Effects <version>\Support Files\Scripts\
